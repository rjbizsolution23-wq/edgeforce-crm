## ⚡ Quick Start

```bash
# 1. Clone
git clone https://github.com/rjbizsolution23-wq/edgeforce-crm.git
cd edgeforce-crm

# 2. Install dependencies
pnpm install

# 3. Copy environment variables
cp .env.example .env.local

# 4. Start local dev
pnpm dev &
```

Open http://localhost:3000 — you're live. 🔥

## 🛠️ Installation

### Prerequisites

| Tool | Version | Install |
|------|---------|---------|
| Node.js | 22+ | `fnm install 22 && fnm use 22` |
| pnpm | 9+ | `npm i -g pnpm` |
| Docker | Latest | [Docker Desktop](https://docker.com) |
| Wrangler | Latest | `pnpm add -g wrangler` |

### Full Setup

```bash
# Install Node via fnm
fnm install 22 && fnm use 22

# Install dependencies (pnpm only — never npm/yarn)
pnpm install

# Start services (Postgres + Redis)
docker-compose up -d

# Configure Cloudflare
wrangler login

# Deploy to preview
wrangler dev
```

## 🚀 Usage

### Dashboard

Access your CRM dashboard at `/dashboard` to manage:
- Contacts and companies
- Deals and pipeline
- Tasks and activities
- Team performance

### API Usage

```typescript
import { api } from '@/lib/api'

// Authenticate
await api.login('email@example.com', 'password')

// Get contacts
const { data } = await api.getContacts()

// Create a deal
await api.createDeal({
  name: 'Enterprise Deal',
  pipelineId: 'pipeline-123',
  stage: 'proposal',
  value: 50000
})
```

### Webhooks

Configure webhooks in Settings > Integrations to receive real-time events:

```typescript
// Example webhook payload
{
  "event": "deal.created",
  "timestamp": "2026-03-30T12:00:00Z",
  "data": {
    "id": "deal_123",
    "name": "Enterprise Deal",
    "value": 50000
  }
}
```

## ⚙️ Configuration

Copy `.env.example` to `.env.local` and fill in your values:

| Variable | Required | Description |
|----------|----------|-------------|
| `NEXT_PUBLIC_APP_URL` | ✅ | Your app's public URL |
| `DATABASE_URL` | ✅ | Supabase PostgreSQL connection |
| `NEXTAUTH_SECRET` | ✅ | Random 32-char secret |
| `STRIPE_SECRET_KEY` | ✅ | Stripe secret key |
| `STRIPE_WEBHOOK_SECRET` | ✅ | Stripe webhook signing secret |
| `NEXT_PUBLIC_SENTRY_DSN` | ✅ | Sentry DSN for error tracking |
| `CF_ACCOUNT_ID` | ✅ | Cloudflare account ID |
| `CF_API_TOKEN` | ✅ | Cloudflare API token |

### Cloudflare Resources (create via wrangler)

```bash
# Create D1 database
wrangler d1 create edgeforce-db

# Create KV namespaces
wrangler kv:namespace create edgeforce-sessions
wrangler kv:namespace create edgeforce-cache
wrangler kv:namespace create edgeforce-ratelimit

# Create R2 bucket
wrangler r2 bucket create edgeforce-assets
```

⚠️ Never commit `.env.local` — it's in `.gitignore` for a reason.

## 📐 Architecture

```
[Client — Next.js 16 App Router]
              ↓
    [Cloudflare Edge / Workers]
              ↓
    [Hono API — 2300+ endpoints]
              ↓
    [Supabase PostgreSQL + Cloudflare D1]
              ↓
    [Cloudflare KV / R2 / Queues / AI]
```

- **Multi-tenant Architecture** — Each tenant has isolated data
- **Edge Computing** — All API runs on Cloudflare Edge for <50ms latency
- **Real-time** — WebSocket support via Durable Objects
- **AI-powered** — Workers AI for lead scoring and chatbots

Full architecture diagram: [docs/ARCHITECTURE.md](./docs/ARCHITECTURE.md)
