## AWS Credits

### Program: AWS Activate

**Estimated Value:** Up to $100,000 in credits

#### How to Apply

1. **Start at:** https://aws.amazon.com/activate/

2. **Three tiers:**
   - Portfolio: $1,000-$10,000 (no verification)
   - Startup: $10,000-$50,000 (incubator required)
   - Accelerate: $50,000-$100,000 (VC required)

#### Eligibility

EdgeForce qualifies for "Accelerate" tier with:
- $2M Seed funding secured
- Built on AWS alternative (can use AWS backup)

#### Use Cases

- S3 for backup storage
- CloudFront CDN (primary Cloudflare)
- Lambda (serverless backup)
- SageMaker (ML training)
